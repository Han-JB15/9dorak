{
    int max = INT_MIN; // int �삎�쓽 理쒖넖媛믪쑝濡� max濡� 媛��젙

    // int i媛� 0遺��꽣 len 誘몃쭔�씪�븣源뚯� 1�뵫 媛믪쓣 利앷��빐 for臾� 諛섎났
    for (int i = 0; i < len; i++) {
        // i踰덉㎏ 諛곗뿴蹂대떎 max媛� �옉�쑝硫�, i踰덉㎏ 諛곗뿴 媛믪쑝濡� max�뿉 ���옣
